# PhotoViewer

A simple iOS App to explore Unsplash API

## Features

- [x] Browse currently popular photos on Unsplash.
- [x] Perform Search query on Unsplash

## Instal

- Clone repo
- run `pod install`
– Cocoapods will ask you to provide UnsplashAccessKey.
– Build and run


## Requirements

- Xcode 10.1
- Swift 4.2.1
- iOS 11 or greater
- CocoaPods 1.5.3
